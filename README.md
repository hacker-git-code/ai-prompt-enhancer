# 🚀 Prompt Enhancer Pro

A **simple and powerful** Chrome extension that automatically enhances your prompts for AI chatbots and tools. No complex UI needed - just click the floating button and your prompts are instantly enhanced!

## ✨ How It Works

1. **Type your prompt** in any AI chat tool (ChatGPT, Claude, Bard, etc.)
2. **Click the floating 🚀 button** that appears on every webpage
3. **Your prompt is automatically enhanced** and inserted back into the chat
4. **Get better AI results** with professional, detailed, creative, or concise prompts

## 🎯 Features

### 🚀 **One-Click Enhancement**
- **Floating Button**: Simple 🚀 button on every webpage
- **Auto-Detection**: Automatically finds chat inputs on AI platforms

## 🛠️ Installation

### Quick Install (Chrome)

1. **Download the Extension**
   ```bash
   git clone https://github.com/your-username/prompt-enhancer-pro.git
   cd prompt-enhancer-pro
   ```

2. **Load in Chrome**
   - Open Chrome and go to `chrome://extensions/`
   - Enable "Developer mode" (top-right toggle)
   - Click "Load unpacked"
   - Select the extension folder

3. **Pin the Extension**
   - Click the puzzle piece icon in Chrome's toolbar
   - Find "Prompt Enhancer Pro" and click the pin icon

## 🚀 Usage

### **Super Simple - Just 2 Steps!**

1. **Type your prompt** in any AI chat tool
   ```
   Example: "help me write a blog post about cooking"
   ```

2. **Click the floating 🚀 button**
   - Your prompt is instantly enhanced
   - The enhanced version replaces your original text
   - You're ready to send!

### **Change Enhancement Style**

1. **Click the extension icon** in your toolbar
2. **Choose your preferred style**:
   - Professional (default)
   - Creative
   - Detailed
   - Concise
3. **Your choice is automatically saved**

## 🐛 Troubleshooting

### **Extension Not Working?**
1. **Refresh the page** where you want to use it
2. **Check if chat input is detected** (look for ✨ indicators)
3. **Ensure you're on a supported AI platform**
4. **Try typing something in the chat box first**

### **Chat Not Detected?**
1. **Wait for page to fully load**
2. **Look for the ✨ indicator** near chat inputs
3. **Try refreshing the page**
4. **Check if the platform is supported**

### **Need to Change Style?**
1. **Click the extension icon** in your toolbar
2. **Select your preferred style**
3. **The change applies immediately**
