# 🚀 Prompt Enhancer Pro

A powerful browser extension that transforms simple prompts into professional, detailed AI instructions for better chatbot and AI tool results.

## ✨ Features

- **Smart Prompt Enhancement**: Transform simple language into powerful AI instructions
- **Multiple Enhancement Styles**: Professional, Creative, Detailed, and Concise
- **Context-Aware Options**: Include examples, formatting instructions, and constraints
- **Quick Templates**: Pre-built templates for common use cases
- **Right-Click Enhancement**: Select any text and enhance it instantly
- **Smart Input Detection**: Automatically finds AI chat input fields
- **Keyboard Shortcuts**: Quick access with Ctrl+Shift+E
- **Floating Toolbar**: Easy access from any webpage
- **Settings Persistence**: Your preferences are saved across sessions

## 🎯 How It Works

1. **Write a Simple Prompt**: Enter your basic request in natural language
2. **Choose Enhancement Style**: Select from Professional, Creative, Detailed, or Concise
3. **Customize Options**: Add examples, formatting, and constraints as needed
4. **Get Enhanced Prompt**: Receive a professionally crafted prompt ready for AI tools
5. **Use Instantly**: Copy, paste, or insert directly into AI chat interfaces

## 📥 Installation

### Chrome/Edge/Brave
1. Download or clone this repository
2. Open Chrome and go to `chrome://extensions/`
3. Enable "Developer mode" in the top right
4. Click "Load unpacked" and select the extension folder
5. The extension icon should appear in your toolbar

### Firefox
1. Download or clone this repository
2. Open Firefox and go to `about:debugging#/runtime/this-firefox`
3. Click "Load Temporary Add-on" and select the `manifest.json` file
4. The extension will be loaded temporarily

## 🚀 Usage

### Basic Enhancement
1. Click the extension icon in your toolbar
2. Enter your simple prompt (e.g., "help me write a blog post")
3. Choose your preferred enhancement style
4. Select additional options (examples, formatting, constraints)
5. Click "✨ Enhance Prompt"
6. Copy or use the enhanced prompt

### Quick Enhancement
1. Select any text on a webpage
2. Right-click and choose "🚀 Enhance as Prompt" or "⚡ Quick Enhance"
3. Get instant enhancement without opening the full interface

### Keyboard Shortcuts
- **Ctrl+Shift+E**: Open the prompt enhancer
- **Right-click on selected text**: Quick enhancement options

### Floating Toolbar
- Look for the floating 🚀 button on web pages
- Click to open the enhancer from anywhere

## 🔧 Technical Details

- **Manifest Version**: 3 (Chrome Extension Manifest V3)
- **Permissions**: Active tab, storage, context menus
- **Content Scripts**: Automatically injected on web pages
- **Service Worker**: Background processing and event handling
- **Storage**: Chrome storage API for settings persistence

## 🐛 Troubleshooting

### Extension Not Working
1. Ensure the extension is properly installed
2. Check if it's enabled in your browser's extension manager
3. Try refreshing the webpage
4. Check the browser console for error messages

### Context Menu Not Appearing
1. Make sure you have text selected
2. Check if the extension has the necessary permissions
3. Try refreshing the page

### Enhanced Prompts Not Inserting
1. Ensure you're on a webpage with input fields
2. Check if the target website allows content script injection
3. Try manually copying and pasting the enhanced prompt
