# 📥 Installation Guide

## 🚀 Quick Start

### Step 1: Download the Extension
1. Clone or download this repository to your computer
2. Extract the files to a folder (e.g., `prompt-enhancer-pro`)

### Step 2: Install in Your Browser

#### 🌐 Chrome / Edge / Brave / Chromium
1. Open your browser and navigate to `chrome://extensions/`
2. Enable **"Developer mode"** in the top-right corner
3. Click **"Load unpacked"**
4. Select the folder containing the extension files
5. The extension should now appear in your extensions list
6. Click the extension icon in your toolbar to start using it

#### 🦊 Firefox
1. Open Firefox and go to `about:debugging#/runtime/this-firefox`
2. Click **"Load Temporary Add-on"**
3. Select the `manifest.json` file from the extension folder
4. The extension will be loaded temporarily
5. **Note**: Firefox requires reloading the extension each time you restart the browser

#### 🧭 Safari
1. Open Safari and go to **Safari > Preferences > Advanced**
2. Enable **"Show Develop menu in menu bar"**
3. Go to **Develop > Show Extension Builder**
4. Click **"+"** and select **"Add Extension"**
5. Choose the extension folder
6. Click **"Run"** to test the extension

## 🔧 Troubleshooting Installation

### Extension Not Appearing
- Make sure you've enabled Developer mode (Chrome/Edge)
- Verify all files are in the same folder
- Check that `manifest.json` is in the root folder
- Try refreshing the extensions page

### Permission Errors
- Ensure the extension has the necessary permissions
- Check if your browser blocks the extension
- Try disabling other extensions temporarily

### Content Script Not Working
- Refresh the webpage after installation
- Check the browser console for error messages
- Verify the extension is enabled

## 📱 Mobile Browsers

**Note**: This extension is designed for desktop browsers. Mobile browsers have limited extension support.

### Chrome Mobile
- Extensions can be installed but functionality may be limited
- Follow the same steps as desktop Chrome

### Firefox Mobile
- Limited extension support
- May not work as expected

## 🔄 Updating the Extension

### Chrome/Edge/Brave
1. Go to `chrome://extensions/`
2. Find "Prompt Enhancer Pro"
3. Click the refresh icon to reload
4. Or remove and reinstall for major updates

### Firefox
1. Go to `about:debugging#/runtime/this-firefox`
2. Click **"Reload"** next to the extension
3. Or remove and reinstall for major updates

## 🎯 Post-Installation Setup

1. **First Use**: Click the extension icon to see the welcome page
2. **Test Enhancement**: Try enhancing a simple prompt
3. **Customize Settings**: Adjust enhancement options to your preference
4. **Try Context Menu**: Select text on any webpage and right-click

## 🚨 Security Notes

- This extension only runs on web pages (not on system files)
- It requires permissions to read page content and inject scripts
- No data is sent to external servers
- All processing happens locally in your browser

## 📞 Getting Help

If you encounter issues during installation:

1. Check the browser console for error messages
2. Verify all files are present and in the correct locations
3. Try disabling other extensions temporarily
4. Check the [README.md](README.md) for troubleshooting tips
5. Ensure your browser is up to date

## 🎉 Success!

Once installed, you should see:
- The 🚀 extension icon in your browser toolbar
- A floating 🚀 button on web pages
- Right-click context menu options for selected text
- The ability to enhance prompts from the popup interface
