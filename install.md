# Installation Guide for Prompt Enhancer

## 🚀 Quick Installation Steps

### For Chrome, Edge, or Brave:

1. **Download the Extension**
   - Download or clone this repository to your computer
   - Extract the files if you downloaded a ZIP

2. **Open Chrome Extensions Page**
   - Open Chrome and go to: `chrome://extensions/`
   - Or click the three dots menu → More tools → Extensions

3. **Enable Developer Mode**
   - Toggle the "Developer mode" switch in the top-right corner

4. **Load the Extension**
   - Click "Load unpacked" button
   - Select the folder containing the extension files
   - Make sure you see `manifest.json` in the folder

5. **Verify Installation**
   - You should see "Prompt Enhancer" in your extensions list
   - The extension icon should appear in your toolbar

### For Firefox:

1. **Download the Extension**
   - Download or clone this repository to your computer

2. **Open Firefox Debugging Page**
   - Go to: `about:debugging`
   - Click "This Firefox" tab

3. **Load Temporary Add-on**
   - Click "Load Temporary Add-on"
   - Select the `manifest.json` file from the extension folder

4. **Note**: Firefox loads extensions temporarily and they'll be removed when you restart the browser

## ✅ Verification

After installation:

1. **Check the Toolbar**: Look for the Prompt Enhancer icon in your browser toolbar
2. **Test on Demo Page**: Open `demo.html` in this folder to test the extension
3. **Test on Real Sites**: Visit ChatGPT, Claude, or other AI chat interfaces

## 🔧 Troubleshooting

### Extension Not Appearing:
- Make sure Developer mode is enabled
- Check that you selected the correct folder (should contain `manifest.json`)
- Try refreshing the extensions page

### Enhance Button Not Showing:
- Ensure the extension is enabled
- Refresh the webpage after installing
- Check browser console for any error messages
- Make sure you're typing at least 10 characters

### Permission Issues:
- The extension needs `activeTab` and `storage` permissions
- These are automatically granted when you load the extension

## 🎯 Testing the Extension

1. **Open the Demo Page**: `demo.html`
2. **Type a Prompt**: Enter at least 10 characters in the textarea
3. **Look for Enhance Button**: A blue "Enhance" button should appear above the input
4. **Click Enhance**: Your prompt will be automatically improved
5. **Try Different Styles**: Use the extension popup to change enhancement styles

## 📱 Mobile Browsers

Note: Browser extensions typically don't work on mobile browsers. This extension is designed for desktop browsers.

## 🔄 Updates

To update the extension:
1. Download the latest version
2. Remove the old version from `chrome://extensions/`
3. Load the new version using "Load unpacked"

## 🆘 Need Help?

If you encounter issues:
1. Check the browser console for error messages
2. Ensure all files are present in the extension folder
3. Try disabling and re-enabling the extension
4. Check that the extension has the necessary permissions
